1.4 逆向复现流程

运行顺序：先生成网络与初始状态，再分别运行同质 MMCA、微观 MMCA 和 MC/QS，最后做策略扫描与绘图。因为作者代码缺失，下面是能够复现论文图型的最小闭环。

  1. 生成超图：设置 n = 1500、d = 3、m = n⟨k⟩/d。对 ⟨k⟩ = 6, 9, 12，对应 m = 3000, 4500, 6000。随机采样不重复的 d 节点子集作为超边。
  2. 初始化：节点以 I(0) 比例感染；通常 I(0)=0.3，Fig. 3 使用 0.2 与 0.8，Fig. 5 使用 0.3 与 0.5。超边活动建议从 Θe(0)=1 开始；rewiring 新边为 Θ0=0.1。
  3. 同质 MMCA：迭代 ρ(t+1) 与 Θ(t+1)，用于 Fig. 2 的理论曲线；稳态用末端平均或固定点求解。
  4. 微观 MMCA：对每个节点计算 qi，对每条超边计算 Φe 与 Θe，可用于比同质理论更接近 MC 的数值验证。
  5. MC/QS：每步同步更新感染和恢复；若进入全健康吸收态，用 QS 历史配置恢复。论文给出历史池大小 50 和概率 0.2，但“0.2 是替换历史池概率还是从吸收态恢复概率”表述不够清楚，复现时应做敏感性检查。
  6. 免疫策略：Random 随机选 ⌊wm⌋ 条活跃超边；TI 按 Φe 降序选；SI 先筛 Θe < θmin。SI1、SI2、SI3 分别对应 θmin=0.2,0.4,0.6。
  7. rewiring：在 t=400 以 TI 去除 w=0.8 超边后，随机或按 (ki+α)/(md+nα) 生成新超边，设置 α=1.0。
  8. 输出：每个参数点记录 ρI(t)、稳态 ρI、Θ、度分布 P(k)、HIT wc，200 次独立运行给均值和误差条。

复现不一致点：SI 排序和物理解释存在张力。算法写的是候选超边按 Θe 降序选前若干；但后文又说 SI1 能更早移除“严重抑制、感染压力高”的低活动超边。若严格降序，预算有限时会先选候选集中活动较高者。较合理的解释是：低阈值 SI1 本身已筛出极低活动边，因此降序影响较小；但如果要严谨复现，应同时测试升序和降序两版。

第二个不一致：论文一处说去激活超边不再参与后续 infection or rewiring events，另一处又说 rewiring 用新超边替换 immunized hyperedges 以保持 m 不变。复现时应把“免疫实验”和“rewiring 实验”分开：前者永久删除传播通道，后者在指定时刻替换删除通道。

二、理论深度解析

2.1 模型层：状态、机制、耦合与时间尺度

状态变量：节点状态 Xi(t) ∈ {S,I} 描述疾病；节点概率 pi(t) 是 MMCA 中对随机状态的边缘概率；超边活动 Θe(t) ∈ [0,1] 是群体交互强度；感染压力 Φe(t) 是群体内部全感染概率。

动态机制：感染不是普通 pairwise SIS，而是“除自己外同组 d-1 人全部感染”才触发。这个全体条件把感染项从线性 ρ 变成 ρd-1，是非连续相变的根源。

耦合方式：疫情通过 Φe 抑制超边活动；活动再通过 Θeβ 抑制感染概率。这是一个负反馈环。η 越大，风险响应越强；γ 越大，恢复社交越快，传播通道越容易重新打开。

时间尺度：论文默认疾病传播、恢复、活动抑制、活动恢复都在同一个离散时间步内发生。这使模型简洁，但也隐藏了真实系统里行为响应滞后和记忆效应。后续最值得拓展的正是这个时间尺度结构。

2.2 动力学层：转移概率、MMCA、平衡与稳定

超边感染压力
Φe(t) ≈ ∏i∈e pi(t)
这是 IBMF 闭包：忽略同组节点感染状态相关性。数学上易处理，物理上会低估簇内相关。
节点避免感染概率
qi(t)=∏e∋i[1-Θe(t)β∏j∈e,j≠ipj(t)]
每条入射超边独立尝试感染；多条超边的“没有感染”概率相乘。
节点概率更新
pi(t+1)=(1-pi)(1-qi)+(1-μ)pi
第一项是健康节点被感染，第二项是感染节点未恢复。
活动更新
Θe(t+1)=Θe(1-ηΦe)+γ(1-Θe)
感染压力越高，活动越被压低；活动越低，自恢复项越强。

平衡条件：同质化后只有两个变量 ρ 和 Θ。稳态活动为 Θ* = γ/(γ+ηρd)，说明只要流行态存在，活动就低于 1；且高 η 或低 γ 会降低有效传播率。

物理含义：这不是简单“传播率变小”。因为 Θ* 本身依赖 ρd，系统出现自抑制：流行越强，群体越退缩，退缩越强，感染又被压低。这种反馈导致同一组参数下可能有无病态与流行态并存。

2.3 数学层：不动点、阈值、Jacobian 与分岔

同质映射可写为：

感染密度映射
F(ρ,Θ)=(1-ρ){1-[1-Θβρd-1]K}+(1-μ)ρ
其中 K=⟨k⟩。
活动映射
G(ρ,Θ)=Θ(1-ηρd)+γ(1-Θ)
二维离散系统的稳定性由 Jacobian 特征值决定。

令 a=1-Θβρd-1，Jacobian 为：

J = [ 1-μ-(1-aK)+(1-ρ)K aK-1Θβ(d-1)ρd-2, (1-ρ)K aK-1βρd-1; -ηdΘρd-1, 1-γ-ηρd ]
这是本文最应该补进 Methods 的对象。局部稳定当两个特征值模长都小于 1。

无病不动点：(ρ,Θ)=(0,1)。当 d≥3，感染项一阶导数为 0，Jacobian 特征值近似为 1-μ 与 1-γ，所以无病态对任何有限 β 线性稳定。传播不能从无限小扰动启动，必须跨过有限振幅阈值。

非线性阈值：流行态由自洽方程给出，等价地将 β 表示为 ρ 的函数：

β(ρ)= 1-[1-μρ/(1-ρ)]1/KΘ*(ρ)ρd-1,   Θ*(ρ)=γγ+ηρd
真正的非线性阈值是 βn=minρ>0 β(ρ)。极小值点满足切触条件 dβ/dρ=0，对应 saddle-node。

分岔行为：对 d≥3，无病态不会经线性失稳变成流行态；流行态成对出现，一个稳定一个不稳定。低于 fold 没有流行态；超过 fold 进入双稳区；系统最终到哪个吸引子取决于初始感染密度和活动响应强度。论文 Fig. 3 中“高初始感染反而清零”正是因为强初值触发强活动抑制，直接把系统推入无病吸引域。

2.4 物理意义层：机制竞争和关键参数

参数数学角色物理意义可发表观察
β感染项线性系数，但乘 ρd-1群体中满足传播条件后的感染强度决定是否越过 saddle-node。
d非线性阶数群体传播所需“共同感染人数”d≥3 是一阶转变来源。
⟨k⟩超边尝试次数指数 K个体参与多少群体提高有效感染机会，降低阈值。
η活动抑制强度风险响应敏感性扩大无病区，增强自隔离。
γ活动恢复强度社会活动恢复速度促进反弹，是控制失败的重要参数。
w超边删除比例外部干预资源HIT 不是静态值，而是与适应反馈共同决定。

最重要的机制竞争：感染传播需要高活动 Θ 和足够高 ρd-1，而活动抑制需要高 ρd。这意味着中等感染可能最危险：足以维持传播，但不足以触发强烈群体退缩；非常高的感染反而可能触发全局活动塌缩。

六、完整模型构建：风险感知-行为记忆自适应超边 SIS

6.1 网络结构与状态定义

网络为 d-uniform hypergraph H(V,E)。每个节点有疾病状态 Xi∈{S,I}；每条超边有活动 Θe∈[0,1]；新增风险记忆状态 Re∈[0,1]，表示群体对过去感染压力的感知风险。

新参数：λ∈(0,1] 是风险记忆更新速率，κ≥0 是风险感知饱和尺度，h(φ)=φ/(κ+φ) 是风险感知函数。若想保持 h(1)=1，可用归一化版本 h(φ)=(1+κ)φ/(κ+φ)，但推导完全类似。

6.2 动态规则与转移概率

风险记忆
Re(t+1)=(1-λ)Re(t)+λh(Φe(t))
群体行为响应的是平滑后的风险，而不是即时感染压力。
活动响应
Θe(t+1)=Θe(t)[1-ηRe(t)]+γ[1-Θe(t)]
当记忆风险高时，即使即时感染下降，群体也会继续低活动。
节点避免感染
qi(t)=∏e∋i[1-Θe(t)β∏j∈e,j≠ipj(t)]
传播层沿用原论文，保证可比较性。
节点 MMCA
pi(t+1)=(1-pi)(1-qi)+(1-μ)pi
新机制只改变活动反馈，不改变疾病状态空间。

6.3 同质 MMCA、稳态方程与阈值

令 K=⟨k⟩，同质变量为 ρ(t)、Θ(t)、R(t)：

ρ(t+1)=(1-ρ){1-[1-Θβρd-1]K}+(1-μ)ρ
Θ(t+1)=Θ[1-ηR]+γ(1-Θ)
R(t+1)=(1-λ)R+λh(ρd)

稳态满足：

R*=h(ρd),   Θ*=γγ+ηR*,   μρ=(1-ρ){1-[1-Θ*βρd-1]K}

因此非线性阈值的参数形式为：

βn=minρ>0 1-[1-μρ/(1-ρ)]1/KΘ*(ρ)ρd-1
与原模型相比，阈值中的 Θ*(ρ) 由 h(ρd) 控制。λ 不改变固定点位置，但改变稳定性和振荡，这是本文可发表的新结论。

6.4 Jacobian、稳定性与分岔

记 a=1-Θβρd-1，三维 Jacobian 在稳态处具有稀疏结构：

J = [ A, B, 0; 0, C, D; E, 0, F ]
A=1-μ-(1-aK)+(1-ρ)KaK-1Θβ(d-1)ρd-2； B=(1-ρ)KaK-1βρd-1； C=1-γ-ηR；D=-ηΘ；E=λh'(ρd)dρd-1；F=1-λ。

特征方程为：

(A-ξ)(C-ξ)(F-ξ)+BDE=0
因为 D<0，记忆回路通过 BDE 改变特征值位置。稳定条件是全部特征值满足 |ξ|<1。

分岔类型：

  • saddle-node：当 ξ=1，等价于稳态曲线发生切触，是原论文的一阶流行阈值。
  • Neimark-Sacker：当一对复特征值穿过单位圆，出现记忆诱导的持续振荡或准周期波动。这是新模型相对原论文的核心可发表点。
  • 双稳与滞后：fold 仍可存在；记忆会改变吸引域边界，使同样初值下出现更长暂态、反弹或延迟清零。

7.2 参数敏感性、相变、热图与误差条

  • 参数敏感性：固定 n=1500,d=3,μ=0.1,K=6,I(0)=0.3，扫描 λ∈[0.02,1]、κ∈[0.001,1]、η∈[0,1]、γ∈[0.005,0.08]。重点看 λ 是否只改变暂态还是也诱发稳定性转换。
  • 相图：绘制 (β,η)、(β,γ)、(λ,η)、(λ,κ) 四组热图。颜色为最终 ρ，边界叠加解析 fold 线和数值 Neimark-Sacker 线。
  • 相变：做上扫和下扫 β，展示滞后环。对每个 λ 比较滞后宽度，证明记忆会改变突变路径。
  • 稳态：报告 ρ*、Θ*、R* 三个稳态量，不只画感染密度。这样读者能看到“风险记忆高但感染低”的恢复滞后状态。
  • 时间演化：近 fold 点、近 Neimark-Sacker 点、远离阈值点各选一组，画 ρ,Θ,R。建议加相位图 ρ-R 或 ρ-Θ。
  • 误差条：MC 每点至少 100-200 次。稳态取最后 20% 时间平均；误差条用跨运行标准误或 bootstrap 置信区间。
  • 有限尺寸检查：用 n=500,1000,1500,3000 检查突变位置和振荡幅度是否稳定，避免把有限尺寸吸收误判为无病态。

7.3 论文 Results 建议结构

  1. Result 1：记忆模型的 MMCA 与 MC 一致，复现原模型阈值并解释新状态变量。
  2. Result 2：风险记忆导致活动恢复滞后，延长或抑制流行暂态。
  3. Result 3：三维 Jacobian 预测 Neimark-Sacker 分岔，MC 观察到持续振荡或准周期波动。
  4. Result 4：记忆参数改变滞后宽度和吸引域，解释高初始感染清零与中等初始感染持久化的机制边界。
  5. Result 5：结合 SI/TI 策略，风险记忆可降低 HIT 或推迟反弹，给出控制启示。

可发表性判断：如果只做 Fig. 2-7 的重复扩展，贡献偏弱；如果能把 Jacobian 特征值、fold 线、Neimark-Sacker 线和 MC 振荡相图连成闭环，就有机会投复杂系统、非线性动力学或应用数学建模方向的较好期刊。
